# CRM Basic

A simple CRM for tracking frontend dev inquiries — built with Next.js, Supabase, and Vercel.

## What it does

- Public contact form for capturing leads
- Admin dashboard to view and manage inquiries
- People directory deduplicated by email
- Inquiry pipeline with stages: new lead → contacted → discovery call → proposal → won / lost

## Stack

- **Framework:** Next.js (App Router)
- **Database:** Supabase (PostgreSQL)
- **Hosting:** Vercel
- **Domain:** crm-axl.vercel.app
